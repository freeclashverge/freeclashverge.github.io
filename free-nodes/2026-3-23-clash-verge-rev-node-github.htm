Surfboard路由器的安装与配置步骤

安装和配置Surfboard路由器并不复杂。只需按照以下步骤操作，您就能轻松完成路由器的设置：

1. 物理连接

首先，确保您已准备好所有必要的硬件，接下来是连接路由器的物理步骤：

  • 将路由器的电源插入电源插座。

  • 使用网线将路由器的WAN端口连接到网络插座或调制解调器。

  • 将电脑或手机连接到路由器的LAN端口（可以使用网线连接，也可以通过无线连接）。

2. 访问管理界面

连接好硬件后，打开浏览器，输入路由器的默认IP地址（一般是192.168.100.1）进行登录。此时，您需要输入默认的用户名和密码（通常可以在路由器的背面找到，或参考用户手册）。

3. 设置Wi-Fi网络

在管理界面中，找到无线网络设置选项。您需要输入以下信息：

  • SSID（无线网络名称）：设置您的网络名称，方便家中的设备识别。

  • 密码：为您的Wi-Fi网络设置一个强密码，确保网络的安全性。

  • 加密方式：建议选择WPA2个人模式，这是目前最安全的加密协议。

4. 保存设置并重启

设置完成后，点击保存按钮，并重启路由器。确保您的Wi-Fi设置已经生效，家中的设备就可以开始连接到网络。

FAQ

Potatso Lite 能否兼容最新的 iOS 系统？
Potatso Lite 更新频率较低，在新版本 iOS 上可能出现兼容性问题。若遇到无法连接或闪退情况，建议升级至其他活跃维护的客户端如 Shadowrocket 或 Quantumult X。
Surfboard 能否同步 iCloud 配置？
可以。Surfboard 支持 iCloud 同步功能，用户可在不同苹果设备上共享同一配置文件。这样即使更换手机或重装系统，也能自动恢复节点与分流设置。
Potatso Lite 节点手动切换有什么优势？
手动切换节点可以让用户根据网络延迟和用途选择最合适的节点，适合在节点不稳定或特殊访问需求下使用。虽然不自动，但可以灵活调整，提高访问效率。
Shadowrocket 的分流规则如何实现广告拦截？
通过配置规则，将广告域名或请求路径匹配到黑名单策略，使请求直接阻断或重定向到本地地址。这样用户浏览网页或使用应用时可自动屏蔽大部分广告内容。
如何优化 Clash DNS 解析速度？
可在配置文件中启用 fake-ip 模式或使用高质量的 DoH 服务（如 Cloudflare、AliDNS）。同时将 `fallback-filter.geoip` 设置为 true，避免国内域名被错误解析到境外，提高解析效率。
V2Ray 策略组如何根据节点延迟自动选择？
策略组会根据节点延迟和丢包率自动选择最优节点。结合路由规则可保证国内流量直连，海外流量走最佳节点，提高访问速度和连接稳定性，优化整体网络体验。
在 Clash 中使用混合代理模式有什么好处？
Clash 的混合代理模式允许同时运行多个代理协议和端口监听，从而实现流量分流。它可以为不同的应用或域名设置不同的代理方式，提高灵活性与连接稳定性，非常适合多协议用户使用。
PassWall2 的规则优先级如何设置？
在 PassWall2 中，规则的匹配顺序按配置文件自上而下执行。建议将精确规则放在前面，通用规则放后面，以确保高优先级的流量优先被正确分流到指定出口。
Mellow TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响指定应用。TUN 更适合全局加速，如游戏、视频和浏览器流量，提高网络稳定性和覆盖范围。
如何在 Windows 上用 Wireshark 分析代理流量以排查问题？
使用 Wireshark 抓取对应网卡流量，结合过滤器（例如 host 或 tcp.port==）分析握手与数据包内容。注意 TLS/QUIC 流量为加密内容，仅能通过元信息（时延、丢包）判断异常，不要尝试解密用户隐私。
